Objective
Conquer the world! Take over every territory on the map to win!

Map
Initial territory allocation is automatically determined at the start of the game. The world consists of 24 territories, split equally among the players. Each player begins with 30 units to distribute throughout their territories.

Instructions
- Select one of the following options:
	- Start game - create new game and select number of players
	- Join game - join new game waiting for certain number of players to begin
	- Continue game - rejoin game to continue playing 
- Follow prompts to enter game
- Click "exit the game" to leave anytime during game play
- Click "logout" to disconnect from server
